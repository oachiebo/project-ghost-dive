- DNS und DHCP konfiguriert

- Erster Domain User m.kusanagi erfolgreich eingeloggt

- 6 Shares mit NTFS Berchtigungen erstellt
- NTFS Berechtigungen gesetzt
- Drive-Mapping Mission M: hinzugefügt
